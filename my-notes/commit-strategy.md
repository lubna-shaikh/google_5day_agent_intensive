
## Examples for GitHub commits
- When you upload a notebook: '`feat: Add day 2 custom tool implementation`

- When you upload your reflections: `docs: Add reflections for day 3`

- When you upload screenshots: `chore: Add screenshots for RAG agent run`

- When you improve a notebook: `refactor: Improve multi-agent task delegation flow`

- When you fix mistakes: `fix: Resolve tool import error in day 4 notebook`
